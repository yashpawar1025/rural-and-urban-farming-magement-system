### Key Features
- **Gradient Overlay**: Ensures text remains readable over any image
- **Relative Positioning**: Allows for future overlay elements if needed
- **Overflow Hidden**: Prevents image bleeding outside rounded corners
- **Parallax Class**: Maintains existing parallax scroll effects where applicable

## Maintenance

To update an image:
1. Search for the page file in `/src/pages/rural/` or `/src/pages/urban/`
2. Locate the hero section `<div>` with `backgroundImage` style
3. Replace the URL in the `url('...')` portion
4. Ensure the new image has similar dimensions and composition
5. Test text readability with the new image
6. Run `npm run lint` to verify no errors
